# 게임 기획서: Flappy Bird (SolidJS + Canvas)

## 1. 개요
플래피 버드는 플레이어가 새를 조종하여 파이프 장애물을 피해 최대한 멀리 이동하는 중독성 있는 아케이드 게임입니다.

## 2. 게임 메커니즘

### 2.1 플레이어 (Bird)
- **속성**: 위치(x, y), 반경, 속도(velocity), 중력(gravity), 점프력(jump strength).
- **동작**:
  - 화면을 클릭하거나 스페이스바를 누르면 위로 점프합니다.
  - 점프하지 않을 때는 중력에 의해 아래로 추락합니다.
  - 화면 상단이나 하단에 닿으면 게임 오버됩니다.

### 2.2 장애물 (Pipes)
- **구조**: 위/아래 두 개의 파이프가 한 쌍을 이루며, 그 사이에 새가 지나갈 수 있는 간격(Gap)이 존재합니다.
- **생성**: 일정 시간 또는 일정 거리 이동 시마다 새로운 파이프 쌍이 화면 오른쪽 끝에서 생성됩니다.
- **이동**: 모든 파이프는 일정한 속도로 왼쪽으로 이동합니다.
- **충돌**: 새의 히트박스가 파이프 영역과 겹치면 게임 오버됩니다.

### 2.3 배경 (Background)
- **스크롤**: 배경 이미지가 왼쪽으로 천천히 이동하여 이동하는 듯한 효과를 줍니다 (Parallax scrolling 고려).

## 3. 물리 엔진 상세
- `velocity += gravity` (매 프레임)
- `position.y += velocity`
- Jump: `velocity = -jump_strength`

## 4. 상태 관리
- **READY**: 게임 시작 전, 클릭 대기 중.
- **PLAYING**: 게임 진행 중, 점수 카운트.
- **GAME_OVER**: 충돌 후 결과 화면, 재시작 대기.

## 5. 점수 시스템
- 파이프 쌍을 완전히 통과할 때마다 점수가 1점씩 올라갑니다.
- 최고 점수(High Score)를 로컬 스토리지에 저장합니다.
